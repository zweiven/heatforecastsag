# heatforecastsag
Code and data for reproducing results in "Little evidence of avoided yield loss 
in US corn when short-term forecasts correctly predict extreme heat"

# Code
code/GenerateHeatForecastResults.R contains all R code to generate figures, 
tables, and summary statistics that appear in the main text.

# Data
data/heatforeacasts_us_corn.csv contains the final dataset used in estimation,
with one row per county and year.

# Results
The results directory contains subdirectories with figures (as .eps files) 
and tables (as .tex files) generated using GenerateHeatForecastResults.R
